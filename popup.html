Math Align Extension

If you ask mathematical questions in right-to-left (RTL) languages like Arabic or Persian, you might notice issues with how math formulas are displayed. This extension helps resolve those issues by setting the direction of KaTeX-rendered math formulas to left-to-right (LTR).

KaTeX is a fast, easy-to-use library for rendering math formulas on the web. To learn more about KaTeX, visit their official website at katex.org.
